# Crypto Price Tracker

A simple cryptocurrency price tracker built with HTML, CSS, and JavaScript. The application displays real-time prices of three popular cryptocurrencies: Bitcoin, Ethereum, and Dogecoin using data fetched from a cryptocurrency API.

## Features

* Live Bitcoin price updates
* Live Ethereum price updates
* Live Dogecoin price updates
* Responsive and clean user interface
* Real-time data fetched from an API

## Technologies Used

* HTML5
* CSS3
* JavaScript
* Cryptocurrency API

## Preview

The application displays the current prices of:

* Bitcoin (BTC)
* Ethereum (ETH)
* Dogecoin (DOGE)

## How It Works

1. JavaScript sends a request to a cryptocurrency API.
2. The API returns the latest cryptocurrency prices.
3. The webpage updates automatically with the fetched data.
4. Users can view current cryptocurrency prices in a simple dashboard.
