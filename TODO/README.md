# Todo List App

A simple Todo List application built with HTML, CSS, and JavaScript.

## Features

- Add new tasks
- Select a due date
- Clean and modern UI
- Responsive design

## Technologies Used

- HTML
- CSS
- JavaScript

## Project Structure

```text
TODO/
├── todolist.html
├── todo-list.css
├── todo-list.js
└── README.md
```

## How to Run

1. Clone the repository

```bash
git clone https://github.com/piyush6705/HTML-CSS-JS.git
```

2. Open the TODO folder

3. Open `todolist.html` in your browser
