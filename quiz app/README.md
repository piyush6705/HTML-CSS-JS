# Quiz App

A simple and interactive Quiz App built using HTML, CSS, and JavaScript. Users can answer multiple-choice questions, receive instant feedback, and track their score throughout the quiz.

## Features

* Multiple-choice questions
* Instant answer validation
* Highlights correct and incorrect answers
* Score tracking
* Responsive design
* Clean and modern user interface
* Restart quiz functionality

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)

## How It Works

1. The quiz starts with the first question.
2. Users select an answer from the available options.
3. The selected answer is checked instantly.
4. Correct answers are highlighted in green.
5. Incorrect answers are highlighted in red.
6. Users can proceed to the next question.
7. At the end of the quiz, the final score is displayed.
